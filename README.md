# Top-10-Indian-Places-To-Visit
📊 Top 10 Indian Places to Visit – Excel Project

Description:
This project presents an analysis of the top 10 tourist destinations in India using Microsoft Excel. It includes details like location, category, cost, ratings, and best time to visit, with interactive charts and dashboards for easy understanding.

Tools Used:
Microsoft Excel, Data Visualization (Bar & Pie Charts), Sorting & Filtering, Conditional Formatting, Basic Data Analysis

Insights:

Historical and cultural places are among the most popular destinations
Higher-rated places often have higher travel costs
Seasonal trends influence travel decisions for hill stations and beaches
Budget-friendly destinations can also have good ratings
Northern and coastal regions attract more tourists
